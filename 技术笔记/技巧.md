4. sort用法

        ```
        people = [[7,0],[4,4],[7,1],[5,0],[6,1],[5,2]]
        people.sort((a, b ) => {
                if(b[0] !== a[0]) {
                    return b[0] - a[0]
                } else {
                    return a[1] - b[1]
                }

            })
        ```
